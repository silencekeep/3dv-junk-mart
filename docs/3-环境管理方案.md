# 环境管理方案

1. **业务后端 (FastAPI)**：使用轻量级 **Python `venv`**，仅安装 Web 和数据库库（如 `sqlalchemy`、`uvicorn`）。
2. **3DGS 训练链 (AI/GPU)**：使用 **Miniconda**，独立创建一个环境 `3dgs_app`，严格隔离底层 CUDA、PyTorch 和 Nerfstudio 依赖。
3. **移动端客户端 (Flutter)**：使用官方 **Flutter SDK**，依赖通过 `pubspec.yaml` 原生管理。
4. **系统级工具 (C++应用与服务)**：**COLMAP**、**FFmpeg**、**Node.js** 和 **MySQL** 直接安装在电脑宿主机，并将工具路径加入**系统环境变量 (PATH)**。

**一句话总结**：前端走官方 SDK，后端用 venv 搞定，AI 必须锁死在 Conda 里，底层工具直接配进系统 PATH。
