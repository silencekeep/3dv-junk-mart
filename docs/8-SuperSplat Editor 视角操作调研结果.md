# SuperSplat Editor 视角操作调研结果

## Orbit Mode

- **双指捏合或扩张**：控制缩放
- **单指滑动**：控制视角围绕物体旋转，如果不移动相机就会始终面向物体，等效于控制物体旋转
- **双指滑动**：控制相机位置的上下左右移动，手指滑动多少，相机移动多少，注意移动相机位置之后单指滑动可能不会始终面向物体了，这是正常的

## Fly Mode

- **双指捏合或扩张**：控制前进和后退，在 Fly Mode 中没有缩放
- **双指平移滑动**：控制相机位置的上下左右移动，不是划一下屏幕才移动一下，而是开始滑动后，只要不松手就持续移动
- **单指滑动**：以相机为中心转动视角，类似所有 3D 游戏的鼠标控制视
